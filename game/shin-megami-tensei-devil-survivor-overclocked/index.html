Summary

Shin Megami Tensei: Devil Survivor Overclocked is playable without issue besides a long black screen with the audio at the beginning. If you wait enough the menu will pop up.

The game requires you to set your CPU Clock Speed to 45% to run near perfectly.

Don’t use save states or load states as it will break the Voice Acting in the game.

The “Summon” option in battles will freeze the game (but not the audio) for a few (usually 3-5) minutes. This doesn’t break the game and causes no further issues after the fact, but will require you to wait quite a while everytime you try to use it.

Summary

Phoenix Wright: Ace Attorney - Dual Destinies has some problems on Citra. The game performs poorly without powerful hardware. Using hardware shader increases FPS, but is heavy on GPU.

Costume and Special Episode DLC is playable on Citra, but early versions of godmode9 would result in DLC being improperly exported (game acknowledges them, but crashes when trying to access either). DLC exported with later versions of godmode9 are fully functional.
